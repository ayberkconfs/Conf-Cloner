# Conf CLONER Elite 🚀

Conf CLONER is a premium, high-performance Discord server cloning tool built with **Electron**, **React**, and **TypeScript**. Designed for power users and server administrators, it provides an elite suite of tools to replicate server structures with surgical precision.

![Application Preview](resources/taskbar_icon.png)

## 💎 Elite Features

- **Full Server Replication:** Clone roles, permissions, categories, text/voice channels, emojis, and stickers.
- **User Token Support:** Operate using Discord User Tokens (Selfbot) for maximum flexibility.
- **Streamer Mode (Privacy):** Instantly hide the application window from OBS, Discord, and other screen capture software to protect your tokens.
- **Token Manager:** Store and verify multiple accounts locally. Switch between verified sessions with a single click.
- **Template Market:** Save your favorite server layouts as templates and deploy them instantly to new destinations.
- **Multi-Language Support:** Full support for **English** and **Turkish** interfaces.
- **Welcome Back System:** Securely remembers your last session and provides a premium "Welcome Back" login experience with your Discord profile.
- **Auto-Update System:** Integrated version tracking via GitHub to ensure you are always running the latest "Elite" build.
- **Glassmorphism UI:** A beautiful, modern "OLED" theme with neon accents and fluid animations.

## 🚀 Installation

You can download the latest stable release from the [Releases](https://github.com/ayberkconfs/Conf-Cloner/releases) page.

1. Download `Conf_CLONER_1.0.0.msi`.
2. Run the installer.
3. Launch **Conf CLONER** from your desktop.

## 🛠️ Development

If you want to build the project from source:

```powershell
# Clone the repository
git clone https://github.com/ayberkconfs/Conf-Cloner.git

# Install dependencies
npm install

# Run in development mode
npm run dev

# Build the MSI installer
npm run dist
```
